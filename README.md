# EFEITOS + UTILITÁRIOS

## Introdução

Neste repositório encontramos a resolução dos exercícios do projecto inicial
de programação em Python. Neste projecto são então propostos os seguintes
exercícios:

- `efeitos.py`: Conjunto de efeitos a aplicar a texto: diagonal, invertido, a deslizar, etc.
- `vcypher.py`: Enciptação de ficheiros com Cifra de Vigènere
- `treep.py`: Clone do comando `tree` muito usado no mundo Unix
- `l2addr.py`: Programa para fornecer endereços MAC a partir de endereços IP
- `anthimio.py`: Programa que processa CSV com dados climatéricos e indica um conjunto de 
  estatísticas relevantes.

Encontra mais informação sobre os exercícios no enunciado do projecto localizado
na directoria **`docs`**.

## Programa `efeitos.py`

Programa em Python para exibir o texto introduzido na linha de comandos de
acordo com determinados "efeitos especiais"
```
    $ efeitos.py [-i INTERVALO] [-d DIM] palavra1 [palavra2] ... [palavraN]
```
Significado das opções:
```
-d : dimensão da linha de texto para o efeito de texto deslizante
-i : intervalo de tempo entre exibições para os efeitos com "movimento"
```

Efeitos são aplicados ao texto que resulta da concatenação de `palavra1`,
`palavra2`, etc. Ver enunciado do projecto em **`docs`** para uma descrição
promenorizada dos efeitos.
